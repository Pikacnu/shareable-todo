## Install

make sure you install bun before start [bun](https://bun.sh)

```sh
bun i
```

## Development

Run the dev server:

```shellscript
bun run dev
```

## Deployment

First, build your app for production:

```sh
npm run build
```

Then run the app in production mode:

```sh
npm start
```
